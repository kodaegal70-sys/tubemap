# 네이버 지도 API 연동 상세 가이드

네이버 지도를 우리 프로젝트에 띄우기 위해서는 **네이버 클라우드 플랫폼(NCP)** 에서 'Client ID'라는 열쇠를 발급받아야 합니다. 아래 과정을 차근차근 따라와 주세요.

---

### 1단계: 네이버 클라우드 플랫폼 가입 및 로그인
1. [네이버 클라우드 플랫폼](https://www.ncloud.com/)에 접속합니다.
2. 회원가입을 진행합니다. (네이버 아이디로 로그인 가능)
3. **결제 수단 등록**: API 사용을 위해 결제 수단(카드) 등록이 필수입니다. 
   > **참고**: `Web Dynamic Map` 서비스는 **월 1,000만 건**까지 무료입니다. 개발 단계에서는 비용이 청구되지 않으니 안심하세요.

### 2단계: Application 등록
1. 로그인 후 우측 상단의 **[Console]** 버튼을 클릭하여 콘솔로 이동합니다.
2. 좌측 메뉴에서 **Services** > **AI·NAVER API** > **Application**을 클릭합니다.
3. **[+ Application 등록]** 버튼을 클릭합니다.
4. **Application 이름**에 `media-loc-map` (또는 원하는 이름)을 입력합니다.
5. **Service 선택** 영역에서 **Maps** 하위의 **[Web Dynamic Map]** 을 체크합니다.
   - ⚠️ *주의: Mobile Dynamic Map이 아닌 **Web Dynamic Map**을 선택해야 합니다.*
6. **서비스 환경 등록** 영역에서:
   - **Web 서비스 URL** 입력창에 `http://localhost:3000` 을 입력합니다.
   - **[+ 추가]** 버튼을 눌러 목록에 추가되었는지 확인합니다.
   - (나중에 도메인이 생기면 그때 추가하면 됩니다. 지금은 로컬 주소만 있으면 됩니다.)
7. **[등록]** 버튼을 클릭하여 완료합니다.

### 3단계: Client ID (API 키) 확인
1. 등록이 완료되면 **Application** 목록 화면으로 돌아옵니다.
2. 방금 만든 `media-loc-map` 항목 아래에 있는 **인증 정보** 버튼(또는 'Client ID'라고 적힌 부분)을 찾습니다.
3. `Client ID` 옆의 긴 문자열(영어+숫자 조합)이 바로 우리가 필요한 키입니다. 이를 **복사**해 두세요.

### 4단계: 프로젝트에 키 입력하기
1. VS Code(또는 사용 중인 에디터)로 돌아옵니다.
2. 프로젝트 최상위 폴더에 있는 **`.env.local`** 파일을 엽니다. (제가 이미 만들어 두었습니다.)
3. 아래와 같이 작성된 부분에서 `YOUR_CLIENT_ID_HERE`를 지우고, 방금 복사한 **Client ID**를 붙여넣습니다.

   **변경 전:**
   ```
   NEXT_PUBLIC_NAVER_MAP_CLIENT_ID=YOUR_CLIENT_ID_HERE
   ```

   **변경 후 예시:**
   ```
   NEXT_PUBLIC_NAVER_MAP_CLIENT_ID=abc123xyz
   ```
4. 파일을 저장합니다 (`Ctrl + S`).

---

### 5단계: 확인하기
1. 터미널(VS Code 하단 등)에서 서버를 실행합니다:
   ```bash
   npm run dev
   ```
2. 웹 브라우저를 열고 `http://localhost:3000` 으로 접속합니다.
3. 흰 화면 대신 **네이버 지도**가 화면에 가득 차게 나오면 성공입니다! 🎉

성공 여부를 알려주시면 다음 단계인 '사이드바'와 '마커' 만들기를 시작하겠습니다.
